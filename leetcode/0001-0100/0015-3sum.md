## 题目描述

给定整数数组 `nums`，找出所有满足 `i != j`、`i != k`、`j != k`，且 `nums[i] + nums[j] + nums[k] == 0` 的三元组 `[nums[i], nums[j], nums[k]]`。

返回的结果中**不能包含重复的三元组**。

### Example 1

```text
Input: nums = [-1,0,1,2,-1,-4]
Output: [[-1,-1,2],[-1,0,1]]
Explanation: 满足条件的三元组是 [-1,-1,2] 和 [-1,0,1]，顺序不限。
```

### Example 2

```text
Input: nums = [0,1,1]
Output: []
```

### Example 3

```text
Input: nums = [0,0,0]
Output: [[0,0,0]]
```

### 约束条件

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`

## 思路分析

### 突破口

排序后固定第一个数，对剩余部分用双指针找两数之和等于负的第一个数，跳过重复元素避免重复三元组。

### 思路拆解

1. **暴力解**：三重循环枚举所有 (i,j,k) 组合，用集合去重，O(n³)，超时。

2. **排序 + 双指针**：先排序 O(n log n)。固定 `i`（从 0 到 n-3），对 `[i+1, n-1]` 区间用双指针 `l=i+1, r=n-1`：
   - 三数之和 < 0 → `l++`
   - 三数之和 > 0 → `r--`
   - 三数之和 == 0 → 记录，然后 `l++` 并跳过重复值，`r--` 并跳过重复值

3. **去重关键**：
   - 外层循环：`nums[i] == nums[i-1]` 时跳过（同一值第二次作为第一个数）。
   - 内层找到结果后：移动 `l` 时跳过与 `l` 相同的值，`r` 同理。

4. **剪枝**：若 `nums[i] > 0`，三数之和不可能为 0（已排序），直接 break。

### 示意图

```text
nums = [-4,-1,-1,0,1,2]（已排序）

i=0: nums[0]=-4, target=4, l=1, r=5
  l=-1, r=2: sum=-4+(-1)+2=-3 < 0 → l++
  l=-1, r=2: sum=-4+(-1)+2=-3 < 0 → l++
  l= 0, r=2: sum=-4+0+2=-2 < 0 → l++
  l= 1, r=2: sum=-4+1+2=-1 < 0 → l++
  l >= r，跳过

i=1: nums[1]=-1, target=1, l=2, r=5
  l=-1, r=2: sum=-1+(-1)+2=0 → 记录[-1,-1,2], l++ 跳过重复, r-- 跳过重复
  l= 0, r=1: sum=-1+0+1=0 → 记录[-1,0,1], l++ r--
  l >= r，跳过

i=2: nums[2]=-1 == nums[1]=-1 → 跳过（去重）
...
结果: [[-1,-1,2],[-1,0,1]]
```

## 复杂度分析

| 解法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 暴力三重循环 | O(n³) | O(n)（结果空间） |
| 排序 + 双指针 | O(n²) | O(n)（排序辅助空间） |

## 代码实现

### Go

```go
import "sort"

// threeSum 找数组中所有和为 0 的不重复三元组
// 参数：nums 整数数组
// 返回：所有满足条件的三元组列表
func threeSum(nums []int) [][]int {
    sort.Ints(nums)
    result := [][]int{}
    n := len(nums)

    for i := 0; i < n-2; i++ {
        // 剪枝：最小的数已大于 0，不可能凑出和为 0 的三元组
        if nums[i] > 0 {
            break
        }
        // 外层去重：跳过与上一个相同的 nums[i]
        if i > 0 && nums[i] == nums[i-1] {
            continue
        }

        l, r := i+1, n-1
        for l < r {
            sum := nums[i] + nums[l] + nums[r]
            if sum < 0 {
                l++
            } else if sum > 0 {
                r--
            } else {
                result = append(result, []int{nums[i], nums[l], nums[r]})
                // 找到后跳过重复值
                for l < r && nums[l] == nums[l+1] {
                    l++
                }
                for l < r && nums[r] == nums[r-1] {
                    r--
                }
                l++
                r--
            }
        }
    }
    return result
}
```

### Java

```java
import java.util.*;

class Solution {
    /**
     * 找数组中所有和为 0 的不重复三元组。
     *
     * @param nums 整数数组
     * @return 所有满足条件的三元组列表
     */
    public List<List<Integer>> threeSum(int[] nums) {
        Arrays.sort(nums);
        List<List<Integer>> result = new ArrayList<>();

        for (int i = 0; i < nums.length - 2; i++) {
            if (nums[i] > 0) break; // 剪枝
            if (i > 0 && nums[i] == nums[i - 1]) continue; // 外层去重

            int l = i + 1, r = nums.length - 1;
            while (l < r) {
                int sum = nums[i] + nums[l] + nums[r];
                if (sum < 0) {
                    l++;
                } else if (sum > 0) {
                    r--;
                } else {
                    result.add(Arrays.asList(nums[i], nums[l], nums[r]));
                    while (l < r && nums[l] == nums[l + 1]) l++; // 内层去重
                    while (l < r && nums[r] == nums[r - 1]) r--; // 内层去重
                    l++;
                    r--;
                }
            }
        }
        return result;
    }
}
```

### Python

```python
class Solution:
    def threeSum(self, nums: list[int]) -> list[list[int]]:
        """
        找数组中所有和为 0 的不重复三元组。

        参数:
            nums: 整数数组
        返回:
            所有满足条件的三元组列表
        """
        nums.sort()
        result = []

        for i in range(len(nums) - 2):
            if nums[i] > 0:
                break  # 剪枝
            if i > 0 and nums[i] == nums[i - 1]:
                continue  # 外层去重

            l, r = i + 1, len(nums) - 1
            while l < r:
                s = nums[i] + nums[l] + nums[r]
                if s < 0:
                    l += 1
                elif s > 0:
                    r -= 1
                else:
                    result.append([nums[i], nums[l], nums[r]])
                    while l < r and nums[l] == nums[l + 1]:
                        l += 1  # 内层去重
                    while l < r and nums[r] == nums[r - 1]:
                        r -= 1  # 内层去重
                    l += 1
                    r -= 1

        return result
```

## 踩坑记录

- **去重逻辑分两层**：外层（固定 `i`）和内层（找到结果后移动 `l/r`）都需要去重，缺一不可。如果只去外层，`[-2,0,0,2,2]` 会重复输出 `[-2,0,2]`。
- **内层去重要先跳后移**：先跳过重复值（while 循环），再执行 `l++; r--` 移动到下一个候选位置，顺序不能颠倒。
- **外层剪枝条件**：`nums[i] > 0` 才能 break（不是 `continue`），因为后续所有 `i` 也都 > 0，完全可以结束。
