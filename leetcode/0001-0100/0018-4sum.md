## 题目描述

给定整数数组 `nums` 和整数 `target`，找出所有满足以下条件的不重复四元组 `[nums[a], nums[b], nums[c], nums[d]]`：

- `a, b, c, d` 是不同的下标。
- `nums[a] + nums[b] + nums[c] + nums[d] == target`

返回结果不能包含重复的四元组（按值去重）。

### Example 1

```text
Input: nums = [1,0,-1,0,-2,2], target = 0
Output: [[-2,-1,1,2],[-2,0,0,2],[-1,0,0,1]]
```

### Example 2

```text
Input: nums = [2,2,2,2,2], target = 8
Output: [[2,2,2,2]]
```

### 约束条件

- `1 <= nums.length <= 200`
- `-10^9 <= nums[i] <= 10^9`
- `-10^9 <= target <= 10^9`

## 思路分析

### 突破口

在 3Sum（排序 + 双指针）的基础上再套一层固定外层数字的循环，变成四重（两层固定 + 双指针），O(n³)。

### 思路拆解

1. **暴力四重循环**：O(n⁴)，太慢。

2. **3Sum 扩展**：排序后，外层两层循环固定 `nums[i]` 和 `nums[j]`（`i < j`），内层对 `[j+1, n-1]` 做双指针求 `nums[l] + nums[r] = target - nums[i] - nums[j]`。

3. **去重**：与 3Sum 类似——外层两个循环各自跳过重复值，内层找到结果后跳过重复值。

4. **注意溢出**：`nums[i] + nums[j] + nums[l] + nums[r]` 在 Java 中可能 int 溢出，需要用 long 计算。

### 示意图

```text
nums = [-2,-1,0,0,1,2]（已排序），target = 0

i=0(-2):
  j=1(-1): target2 = 0-(-2)-(-1) = 3, l=2, r=5
    l=0, r=2: sum=3 → 记录[-2,-1,0... 等等，0+2=2≠3
    实际: l=2(0), r=5(2): 0+2=2 < 3 → l++
    l=3(0), r=5(2): 0+2=2 < 3 → l++
    l=4(1), r=5(2): 1+2=3 → 记录[-2,-1,1,2]
    ...
  j=2(0): target2=2, l=3, r=5
    l=3(0), r=5(2): 0+2=2 → 记录[-2,0,0,2]
    ...
i=1(-1):
  j=2(0): target2=1, l=3, r=5
    l=3(0), r=5(2): 0+2=2 > 1 → r--
    l=3(0), r=4(1): 0+1=1 → 记录[-1,0,0,1]
...
```

## 复杂度分析

| 解法 | 时间复杂度 | 空间复杂度 |
|------|-----------|-----------|
| 排序 + 双层循环 + 双指针 | O(n³) | O(n)（排序） |

## 代码实现

### Go

```go
import "sort"

// fourSum 找所有和为 target 的不重复四元组
// 参数：nums 整数数组，target 目标和
// 返回：所有满足条件的四元组列表
func fourSum(nums []int, target int) [][]int {
    sort.Ints(nums)
    result := [][]int{}
    n := len(nums)

    for i := 0; i < n-3; i++ {
        // 外层第一重去重
        if i > 0 && nums[i] == nums[i-1] {
            continue
        }
        for j := i + 1; j < n-2; j++ {
            // 外层第二重去重
            if j > i+1 && nums[j] == nums[j-1] {
                continue
            }
            l, r := j+1, n-1
            for l < r {
                sum := nums[i] + nums[j] + nums[l] + nums[r]
                if sum < target {
                    l++
                } else if sum > target {
                    r--
                } else {
                    result = append(result, []int{nums[i], nums[j], nums[l], nums[r]})
                    for l < r && nums[l] == nums[l+1] {
                        l++
                    }
                    for l < r && nums[r] == nums[r-1] {
                        r--
                    }
                    l++
                    r--
                }
            }
        }
    }
    return result
}
```

### Java

```java
import java.util.*;

class Solution {
    /**
     * 找所有和为 target 的不重复四元组。
     *
     * @param nums   整数数组
     * @param target 目标和
     * @return 所有满足条件的四元组列表
     */
    public List<List<Integer>> fourSum(int[] nums, int target) {
        Arrays.sort(nums);
        List<List<Integer>> result = new ArrayList<>();
        int n = nums.length;

        for (int i = 0; i < n - 3; i++) {
            if (i > 0 && nums[i] == nums[i - 1]) continue;
            for (int j = i + 1; j < n - 2; j++) {
                if (j > i + 1 && nums[j] == nums[j - 1]) continue;
                int l = j + 1, r = n - 1;
                while (l < r) {
                    // 用 long 避免四数相加溢出 int
                    long sum = (long) nums[i] + nums[j] + nums[l] + nums[r];
                    if (sum < target) {
                        l++;
                    } else if (sum > target) {
                        r--;
                    } else {
                        result.add(Arrays.asList(nums[i], nums[j], nums[l], nums[r]));
                        while (l < r && nums[l] == nums[l + 1]) l++;
                        while (l < r && nums[r] == nums[r - 1]) r--;
                        l++; r--;
                    }
                }
            }
        }
        return result;
    }
}
```

### Python

```python
class Solution:
    def fourSum(self, nums: list[int], target: int) -> list[list[int]]:
        """
        找所有和为 target 的不重复四元组。

        参数:
            nums:   整数数组
            target: 目标和
        返回:
            所有满足条件的四元组列表
        """
        nums.sort()
        result = []
        n = len(nums)

        for i in range(n - 3):
            if i > 0 and nums[i] == nums[i - 1]:
                continue
            for j in range(i + 1, n - 2):
                if j > i + 1 and nums[j] == nums[j - 1]:
                    continue
                l, r = j + 1, n - 1
                while l < r:
                    s = nums[i] + nums[j] + nums[l] + nums[r]
                    if s < target:
                        l += 1
                    elif s > target:
                        r -= 1
                    else:
                        result.append([nums[i], nums[j], nums[l], nums[r]])
                        while l < r and nums[l] == nums[l + 1]:
                            l += 1
                        while l < r and nums[r] == nums[r - 1]:
                            r -= 1
                        l += 1
                        r -= 1

        return result
```

## 踩坑记录

- **Java int 溢出**：四个 `10^9` 相加超出 int 范围（约 4×10^9 > 2.1×10^9），必须用 `long` 计算中间结果。
- **第二层去重条件**：`j` 的去重是 `j > i+1`（不是 `j > 0`），因为 `j` 的起点是 `i+1`，若写 `j > 0` 会错误跳过合法的 `j = i+1` 情况。
- **不要加剪枝 `nums[i] > target`**：target 可能为负数，`nums[i]` 为负时仍可能凑成 target，不能简单剪枝（除非加上 `target >= 0` 的前提条件）。
