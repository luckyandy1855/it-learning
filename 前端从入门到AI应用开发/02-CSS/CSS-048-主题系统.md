# 主题系统（CSS-048）

## 🎯 本节学习目标

- 理解主题系统由设计 token、语义变量和组件变量组成。
- 掌握用 `data-theme` 切换主题的基本方式。
- 能够为 AI 应用设计可扩展的主题结构。

---

## 📖 什么是主题系统

主题系统是一套统一管理颜色、间距、圆角、字体和状态样式的机制。

它不仅服务暗色模式，也可以支持品牌主题、用户自定义主题、不同工作区主题。

在 AI 应用中，主题系统能让聊天、知识库、仪表盘、设置页保持一致，也方便未来做多品牌或个人偏好。

---

## 🧠 原理讲解

一种常见结构是：

```text
基础 token：--blue-600、--gray-100
语义 token：--color-primary、--color-surface
组件 token：--chat-user-bg、--button-bg
```

主题切换可以通过 HTML 属性控制：

```css
[data-theme="dark"] {
  --color-bg: #0f172a;
  --color-text: #e5e7eb;
}
```

只要组件使用变量，切换 `data-theme` 后视觉就会同步变化。

---

## 🏗 基本结构

```html
<html data-theme="light">
<style>
  :root {
    --color-bg: #ffffff;
  }

  [data-theme="dark"] {
    --color-bg: #111827;
  }
</style>
```

## ✅ 完整代码

下面用 `data-theme` 实现 AI 应用主题切换结构：

```html
<!doctype html>
<html lang="zh-CN" data-theme="dark">
<head>
  <meta charset="UTF-8">
  <title>主题系统示例</title>
  <style>
    :root {
      --color-bg: #f8fafc;
      --color-surface: #ffffff;
      --color-text: #111827;
      --color-muted: #6b7280;
      --color-primary: #2563eb;
      --color-border: #e5e7eb;
      --chat-user-bg: #dbeafe;
      --chat-assistant-bg: #f9fafb;
    }

    [data-theme="dark"] {
      --color-bg: #0f172a;
      --color-surface: #111827;
      --color-text: #e5e7eb;
      --color-muted: #9ca3af;
      --color-primary: #60a5fa;
      --color-border: #374151;
      --chat-user-bg: #1e3a8a;
      --chat-assistant-bg: #1f2937;
    }

    * {
      box-sizing: border-box;
    }

    body {
      margin: 0;
      padding: 32px;
      font-family: system-ui, sans-serif;
      background: var(--color-bg);
      color: var(--color-text);
    }

    .app {
      max-width: 720px;
      margin: 0 auto;
      padding: 24px;
      border: 1px solid var(--color-border);
      border-radius: 8px;
      background: var(--color-surface);
    }

    .message {
      padding: 12px 14px;
      border-radius: 8px;
      line-height: 1.8;
    }

    .message.user {
      background: var(--chat-user-bg);
    }

    .message.assistant {
      background: var(--chat-assistant-bg);
      border: 1px solid var(--color-border);
    }

    .theme-note {
      color: var(--color-muted);
      font-size: 14px;
    }
  </style>
</head>
<body>
  <main class="app">
    <p class="theme-note">当前页面通过 data-theme="dark" 使用暗色主题。</p>
    <p class="message user">主题系统和暗色模式有什么区别？</p>
    <p class="message assistant">暗色模式是一种主题；主题系统是一套管理多种主题的机制。</p>
  </main>
</body>
</html>
```

---

## 🔍 逐行解析

`:root` 定义默认主题变量。

`[data-theme="dark"]` 覆盖同名变量，形成暗色主题。

组件全部使用变量，不直接写死主题颜色。

`--chat-user-bg` 和 `--chat-assistant-bg` 是组件级 token，用来表达聊天气泡的特定语义。

---

## 🌐 浏览器表现

示例中 HTML 写了 `data-theme="dark"`，因此页面使用暗色主题。把它改成 `light` 或删除属性，会回到默认明色主题。

真实项目里，可以用 JavaScript 根据用户选择修改 `document.documentElement.dataset.theme`。

---

## 📦 常见属性 / API

| 属性/API | 类型 | 说明 | 示例 |
|---------|------|------|------|
| `data-theme` | HTML 属性 | 标记当前主题 | `data-theme="dark"` |
| 属性选择器 | CSS 选择器 | 根据主题覆盖变量 | `[data-theme="dark"]` |
| design token | 设计概念 | 可复用设计值 | `--color-primary` |
| semantic token | 设计概念 | 按用途命名的变量 | `--color-surface` |
| dataset | DOM API | 修改 data 属性 | `dataset.theme = "dark"` |

---

## ⭐⭐⭐⭐⭐ 必学重点

- 主题系统比暗色模式更大，暗色模式只是主题之一。
- 主题切换的核心是覆盖语义变量。
- 组件不应直接依赖具体颜色值。

---

## ⚠️ 易错点

- 易错点1：组件里大量写死颜色。正确写法：使用语义变量。
- 易错点2：变量层级混乱。正确写法：区分基础 token、语义 token、组件 token。
- 易错点3：主题只覆盖背景和文字。正确写法：边框、状态、气泡、代码块都要覆盖。

---

## 💡 最佳实践

- 先设计语义变量，再写组件。
- 用户可切换主题时，用 `data-theme` 管理明确状态。
- 主题变量要覆盖完整 UI 状态，包括 hover、error、success、selected。

---

## 🚀 AI 应用场景

AI 工具可能服务不同用户：写作、代码、运营、教育。主题系统可以让同一套组件适配不同工作区和品牌外观。

```css
[data-theme="writer"] {
  --color-primary: #7c3aed;
}

[data-theme="developer"] {
  --color-primary: #2563eb;
}
```
