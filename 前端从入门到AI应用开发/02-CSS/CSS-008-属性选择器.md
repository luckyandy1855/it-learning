# 属性选择器（CSS-008）

## 🎯 本节学习目标

- 掌握属性选择器的常见写法，例如 `[disabled]`、`[type="email"]`。
- 理解属性选择器适合根据 HTML 属性和状态选择元素。
- 能够为 AI 表单、模型选项和数据状态编写更灵活的样式。

---

## 📖 什么是属性选择器

属性选择器根据元素是否拥有某个属性，或属性值是否符合条件来匹配元素。

常见写法包括 `[disabled]`、`[type="text"]`、`[data-state="loading"]`、`[href^="https"]`。

在 AI 应用中，属性选择器很适合处理表单控件、模型状态、数据属性和链接来源，不一定每种状态都额外写一个类名。

---

## 🧠 原理讲解

最简单的属性选择器：

```css
button[disabled] {
  opacity: 0.6;
}
```

表示选择所有带 `disabled` 属性的按钮。

匹配具体属性值：

```css
[data-state="loading"] {
  cursor: wait;
}
```

表示选择所有 `data-state` 等于 `loading` 的元素。

---

## 🏗 基本结构

```html
<style>
  input[type="search"] {
    border-color: #2563eb;
  }

  [data-state="loading"] {
    opacity: 0.7;
  }
</style>

<input type="search" placeholder="搜索知识库">
<button data-state="loading">生成中</button>
```

## ✅ 完整代码

下面用属性选择器处理 AI 表单状态：

```html
<!doctype html>
<html lang="zh-CN">
<head>
  <meta charset="UTF-8">
  <title>属性选择器示例</title>
  <style>
    body {
      margin: 0;
      padding: 32px;
      font-family: system-ui, sans-serif;
      background: #f8fafc;
      color: #111827;
    }

    .model-form {
      max-width: 720px;
      margin: 0 auto;
      padding: 24px;
      background: #ffffff;
      border: 1px solid #e5e7eb;
      border-radius: 8px;
    }

    input[type="search"],
    textarea[data-role="prompt"] {
      box-sizing: border-box;
      width: 100%;
      padding: 12px;
      border: 1px solid #d1d5db;
      border-radius: 6px;
      font: inherit;
    }

    textarea[data-role="prompt"] {
      min-height: 120px;
      margin-top: 12px;
    }

    button[data-state="idle"] {
      background: #2563eb;
      color: #ffffff;
    }

    button[data-state="loading"] {
      background: #9ca3af;
      color: #ffffff;
      cursor: wait;
    }

    button[disabled] {
      opacity: 0.7;
    }

    .model-card[data-selected="true"] {
      border-color: #2563eb;
      background: #eff6ff;
    }

    .model-card {
      margin-top: 12px;
      padding: 12px;
      border: 1px solid #e5e7eb;
      border-radius: 6px;
    }
  </style>
</head>
<body>
  <form class="model-form">
    <h1>AI 模型配置</h1>
    <input type="search" placeholder="搜索知识库">
    <textarea data-role="prompt">请根据资料生成摘要。</textarea>

    <div class="model-card" data-selected="true">Claude Sonnet</div>
    <div class="model-card" data-selected="false">GPT 推理模型</div>

    <button type="button" data-state="idle">开始生成</button>
    <button type="button" data-state="loading" disabled>生成中</button>
  </form>
</body>
</html>
```

---

## 🔍 逐行解析

`input[type="search"]` 只选择搜索输入框，不影响其他 input。

`textarea[data-role="prompt"]` 选择带有指定数据角色的 Prompt 文本框。

`button[data-state="idle"]` 和 `button[data-state="loading"]` 根据状态属性区分按钮样式。

`.model-card[data-selected="true"]` 选择被选中的模型卡片。

`button[disabled]` 选择所有禁用按钮，统一降低透明度。

---

## 🌐 浏览器表现

搜索框和 Prompt 文本框拥有统一输入样式。被选中的模型卡片会变成浅蓝背景。生成中的按钮显示灰色，并呈现等待状态。

在 DevTools 中修改 `data-state` 或 `data-selected` 的值，可以立即看到对应样式变化。

---

## 📦 常见属性 / API

| 属性/API | 类型 | 说明 | 示例 |
|---------|------|------|------|
| `[disabled]` | 属性选择器 | 选择带 disabled 属性的元素 | `button[disabled]` |
| `[type="search"]` | 属性值选择器 | 选择属性值完全匹配的元素 | `input[type="search"]` |
| `[data-state="loading"]` | 属性值选择器 | 选择指定状态元素 | `[data-state="loading"]` |
| `[href^="https"]` | 前缀匹配 | 选择以指定值开头的属性 | `a[href^="https"]` |
| `[class~="active"]` | 单词匹配 | 匹配空格分隔列表中的词 | `[class~="active"]` |

---

## ⭐⭐⭐⭐⭐ 必学重点

- 属性选择器可以根据 HTML 属性和值选择元素。
- `data-*` 属性配合属性选择器很适合表达 UI 状态。
- 属性选择器不应替代所有类名，组件基础样式仍然推荐类选择器。

---

## ⚠️ 易错点

- 易错点1：属性值拼写不一致。正确写法：`data-state="loading"` 和 CSS 中的值必须完全一致。
- 易错点2：把所有样式都写成属性选择器。正确写法：基础样式用类，状态样式可用属性。
- 易错点3：误用 `[type=text]`。正确写法：建议写成 `[type="text"]`，更清晰。

---

## 💡 最佳实践

- 表单类型差异适合用 `[type="..."]`。
- 组件状态可以使用 `data-state`、`data-selected`、`aria-expanded` 等属性。
- 对可访问性状态，优先复用真实属性，例如 `[disabled]`、`[aria-current="page"]`。

---

## 🚀 AI 应用场景

AI 应用中常见状态包括 `idle`、`loading`、`error`、`done`。属性选择器可以直接基于 `data-state` 控制视觉。

```css
.generation[data-state="loading"] {
  opacity: 0.7;
}

.generation[data-state="error"] {
  border-color: #ef4444;
  background: #fef2f2;
}

.model-option[aria-checked="true"] {
  outline: 2px solid #2563eb;
}
```

3. [AI 场景题] 为模型卡片设计 `data-selected="true"` 的选中样式。

| 知识点 | 一句话总结 |
|--------|-----------|
| 属性选择器 | 根据属性或属性值匹配元素 |
